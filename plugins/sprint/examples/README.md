# Sprint Examples

This folder contains examples of real Sprint structures.

## Example List

### 1. simple-api/

Sprint example for a simple API project

- 2 Features
- Single agent workflow
- Basic workflow

### 2. multi-agent/

Sprint example for multi-agent collaboration

- 3 Features
- 2 agents (#danny, #agent-1)
- Concurrent work, dependencies, handoffs

## Usage

Reference these examples to create your own Sprint:

```bash
# 1. Copy example
cp -r examples/simple-api sprints/my-sprint

# 2. Modify content
# Edit sprints/my-sprint/BACKLOG.md

# 3. Start working
@sprints/my-sprint/INSTRUCTION.md #your-name
```

## Note

- Examples are for learning purposes
- For real projects, use `/sprint:init` to create fresh
