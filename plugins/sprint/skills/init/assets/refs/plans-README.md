# Task Plans

Execution plans for complex tasks.

## Naming Convention

`F{n}-T{m}-{task-name}.md`

Examples:
- `F1-T1-login-api.md`
- `F2-T3-payment-webhook.md`

## When to Create

- For tasks requiring multiple steps
- When approach needs documentation before coding
- Optional for simple, straightforward tasks
