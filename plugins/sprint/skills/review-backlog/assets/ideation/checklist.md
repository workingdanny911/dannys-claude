# Ideation Review Checklist

> 아이디어/탐색/브레인스토밍 Task/Feature 검토용

---

## 1. 목표 달성 (Goal Achievement)

- [ ] 정의된 질문/문제에 답변했는가?
- [ ] 탐색 범위가 충분했는가?
- [ ] 결론이 명확한가?
- [ ] 다음 단계가 정의되었는가?

---

## 2. 탐색 품질 (Exploration Quality)

- [ ] 충분한 대안을 검토했는가?
- [ ] 각 옵션의 장단점을 분석했는가?
- [ ] 근거/출처가 명시되어 있는가?
- [ ] 가정(assumption)이 명확히 기술되어 있는가?

---

## 3. 실현 가능성 (Feasibility)

- [ ] 제안된 아이디어가 현실적인가?
- [ ] 필요한 리소스가 파악되었는가?
- [ ] 제약 조건이 고려되었는가?
- [ ] 리스크가 식별되었는가?

---

## 4. 문서화 (Documentation)

- [ ] 결정 사항이 기록되었는가?
- [ ] 기각된 대안도 기록되었는가? (이유 포함)
- [ ] 향후 참고할 수 있는 형태인가?
- [ ] 관련 자료/링크가 포함되어 있는가?

---

## 5. 다음 단계 명확성 (Next Steps)

- [ ] 액션 아이템이 정의되었는가?
- [ ] 우선순위가 명확한가?
- [ ] 담당자/일정이 명시되어 있는가? (해당 시)
- [ ] 의존성이 파악되었는가?
